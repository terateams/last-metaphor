# 《遗迹：最后的隐喻》统一时间轴

> **用途**：作为所有时间相关描述的权威参考，确保各文件（`worldview.md`、`structure-7parts.md`、`core-logic-chain.md`、角色档案）的时间线一致。
> **坐标系**：以 **"遗迹出土"为第 0 年**，向前用负数（如 -10 年），向后用正数（如 +25 年）。

---

## 一、时间坐标系定义

| 符号 | 含义 |
|------|------|
| Y-n | 遗迹出土前 n 年 |
| Y0 | 遗迹出土年（第一部开始） |
| Y+n | 遗迹出土后 n 年 |

**故事总跨度**：约 **60 年**（Y-10 至 Y+50），覆盖 2–3 代人。
