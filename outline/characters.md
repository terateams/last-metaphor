# 角色系统总览

> 用途：给每个角色标清 **功能（管哪条线）+ 弧线（从哪到哪）**，写作时防止人物只剩“工具人”。

本文件现在作为 **索引与综述** 使用，具体人物画像拆分至子目录 `outline/characters/` 中：

- 核心主角团：
  - `characters/lin-qiao.md`
  - `characters/zhou-man.md`
  - `characters/xu-qi.md`
  - `characters/du-heng.md`
  - `characters/han-ce.md`
  - `characters/shen-yi.md`
  - `characters/luo-qi.md`
  - `characters/pi-sentinel.md`（Π-哨兵，古人回声）
- 重要配角：`characters/supporting-cast.md`

> 铁规则：每个有名字的人，必须有一个 **不可替代功能**，否则只当功能性人群处理。

---

## 一、层级与配额速览

- **A 层：主角团（7–9 人）** —— 有完整弧线，分别人格化 A/B/C 三条主线与普通人线。
- **B 层：重要配角（12–18 人）** —— 每人只负责 1–2 个关键任务，不写成长篇心理史。
- **C 层：功能性人物** —— 不命名或只出现一次，用来展示制度触角和日常质感。

配额建议：

- 有完整弧线的角色：≤ 9 人（已基本占满，慎重新增）；
- 反复出现的配角：≤ 18 人（集中放入 `supporting-cast.md` 管理）；
- 全书有名有姓总数：尽量 ≤ 35。

检查新角色时，可用以下四连问过滤：

1. 他能推动一个关键转折吗？
2. 他能展示一个系统的真实运作吗？
3. 他能制造一次不可逆代价吗？
4. 他能带来一个新的视角（阶层/职业/价值）吗？

如果四项都是否：

- 不给名字；
- 只作为“司机 / 官员 A / 技师 B”级别功能人物出现；
- 不要在多个章节反复出现，以免读者误以为他有弧线。

## 二、主角团快速索引

详细画像见各自文件，这里只保留一句话功能标签 + 弧线起终点，写作时当“导航目录”用。

1. **林峤（考古异常侦探）** —— `characters/lin-qiao.md`
   - 功能：把散落的异常、遗迹碎片、观测数据整合成可推演的规则假说；在读者视角中承接“求真冲动”，对抗官方稳态叙事。
   - 弧线：从“只想把谜题解完的科学家”，走向“宁可牺牲答案，也要守住可能性的守门人”。

2. **周曼（暗物质通信工程）** —— `characters/zhou-man.md`
   - 功能：把“古人回声”工程化为可读信号，是 C 线的主要技术执行者，也是因果债务的第一责任人。
   - 弧线：从“所有信号都值得解码”的乐观主义者，到“每次读取都是加杠杆”的债务承受者，最终在“关口 vs 继续打开”之间抉择。

3. **许栖（语言学 / 符号学，隐喻保护）** —— `characters/xu-qi.md`
   - 功能：证明语言贫乏是制度性收敛的结果，设计/参与“语言疫苗”，维护富语言地下网。
   - 弧线：从“清晰表达是福音”出发，在家庭/课堂里目睹“词汇减少 = 关系死亡”，最终转身做秘密播种者。

4. **杜衡（指标治理官，B 线人格化）** —— `characters/du-heng.md`
   - 功能：以“文明免疫系统”之名，把 SAI 等指标制度化，是指标崇拜的具象化身。
   - 弧线：从真诚的秩序维护者，走向意识到“自己设计的指标在制造压迫”，最终在“拆解 / 篡改 / 黑化押注”之间选边。

5. **韩策（安全系统 / 军方，加速器）** —— `characters/han-ce.md`
   - 功能：掌握武装与安全权限，把“复刻古人技术”当成优势筹码，以“国家安全、先发优势”为理由施压 A/B/C 线所有人。
   - 弧线：从经典现实主义博弈者，到在同步灾难与因果债务面前怀疑“胜利”的意义，最终要么踩刹车，要么压死骆驼。

6. **沈仪（普通人 A 线：家庭 / 教育）** —— `characters/shen-yi.md`
   - 功能：在教育/家庭一线，把 A 线写进作文、隐喻扣分与情绪压扁的细节里，是“语言贫乏第一受害者”的最近镜头。
   - 弧线：在“配合规范 / 保护孩子”之间反复摇摆，最终要决定是加入富语言地下网，还是在体制内温和抵抗。

7. **罗启（普通人 B 线：一线岗位）** —— `characters/luo-qi.md`
   - 功能：在物流/维修/医护等一线岗位，把 B/C 线的荒诞具象化——“系统显示一切正常，但现场在崩溃”。
   - 弧线：从“信任系统”的螺丝钉，到被迫在“保指标”与“救人/救现场”之间做违纪抉择。

8. **Π-哨兵（古人回声，非人角色）** —— `characters/pi-sentinel.md`
   - 功能：作为暗物质层中的协议残片，提供“关系式、阈值、否定”，不提供完整答案，是 Reset 机制的“口头遗嘱”代言人。
   - 弧线：类弧线（非传统人格）：从被当作工具调用，到被主角们意识为“边界的化身”，出场次数极少但每次都带来结构性代价。

> 写章节前，先锁定本章“主视角”在上述哪一位；连续多章只有精英线时，务必插入沈仪/罗启视角，检查“正确但不可活”的日常后果是否还在。

## 三、重要配角分区

重要配角的完整列表与画像，集中维护在 `characters/supporting-cast.md`。这里只列出必需席位与功能坑位，便于规划：

- 考古队负责人 —— 和林峤在“证据链 vs 进度”上形成长期冲突；
- 材料学专家 —— 推动“可复刻”诱惑与技术升级的关键推手；
- 舆情 / 媒体官 —— 幸福麻醉与叙事一致性的执行者；
- 教育系统督导 —— SAI 进入学校的落地黑手；
- 医院 / 急救系统管理者 —— “正确但不可用”制度化的见证人；
- AI 布道者 / 产品负责人 —— 把精确当美德的代言人；
- 诗歌黑市中间人 —— 语言疫苗地下渠道的运营者；
- 举报者 / 合规官 —— 把恐惧转换为流程与数据；
- 家属 / 伴侣（分别绑定沈仪、罗启或精英线角色）—— 让决策带情感代价；
- 年轻学生 / 孩子 —— 让语言坍缩的代价可见可痛；
- 旧档案管理员 / 历史修订员 —— 历史被“优化”与删节的操作人；
- 委员会法学 / 伦理专家 —— 第六部清算时的道德对撞代表。

新增配角时，优先填补上述“坑位”，确保每个人只完成 1–2 次关键任务，不要给他们完整成长弧线。

## 四、“古人”、功能性人物与区域设定

- 古人回声（Π-哨兵等）出场次数要 **极其克制**：
  - 每一次出场都必须让世界付出某种 **可感知代价**（概率异常、政策后果、个人牺牲等）。

- 功能性人物（司机、官员 A、技师 B 等）：
  - 只负责展示制度的触角与日常质感；
  - 不要给清晰姓名与复杂动机，避免读者误判为“有弧线”；
  - 尽量一次性使用，避免在多个部分反复出现。

- 区域与国别（对所有角色适用）：
   - 可以暗示角色来自不同语言 / 文化背景，但避免点名现实国家与具体城市；
   - 优先使用“某区块 / 某联盟 / 某委员会”等中性治理单元，而不是现实国名；
   - 对读者来说，重要的是他在 **哪套制度里的哪个岗位**，而不是拿哪本护照。

后续在 `characters/` 子目录内补写或更新画像时，只需要保证：

1. 不突破 A 层 9 人、B 层 18 人的软上限；
2. 每个有名字的角色，其“不可替代功能”在对应文件开头一屏内就能看明白；
3. 本文件始终保持为 **轻量导航 + 配额护栏**，而不是堆满细节的设定坟场。
