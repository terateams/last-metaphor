# 许栖｜语言学 / 符号学家（隐喻保护者）

- **层级**：A 层主角团
- **绑定主线**：A 线核心（语言压缩与隐喻存亡）
- **不可替代功能**：
  - 证明“语言贫乏”是制度性收敛，而非自然退化；
  - 设计或参与设计“语言疫苗计划”，维护富语言地下网络；
  - 在设定层面把“清晰崇拜”从理念拆解为可操作的制度与技术细节。

---

## 一、人设基底

- **性别**：女。
- **职业与背景**：
  - 语言学 / 符号学 / 认知科学交叉学者；
  - 早期参与 SAI 指标设计的咨询工作，亲手写过“标准表达指南”。
- **性格基调**：
  - 情绪底色：外表温和、语速偏慢，很少直接发火，但情绪层次丰富且持续时间长；
  - 思维方式：习惯从多个语境和历史脉络看同一句话，对“一句话就说清”的要求本能排斥；
  - 社交习惯：偏内向，社交场合多处于观察者位置，更擅长一对一深聊而非大场合发言；
  - 职业姿态：对自己的专业判断有自信，但在制度场合往往被迫把复杂观点压缩成几条要点；
  - 自我矛盾：一边享受拆解语言、追求表达精度，一边逐渐意识到自己也参与了“清晰崇拜”的建构。
- **核心信念**：
  - 起点：相信“清晰表达能减少误解与暴力”；
  - 动摇点：发现“清晰”被机器与制度接管后，会反过来消灭复杂人。

---

## 二、人物弧线

### 起点：清晰崇拜的技术合作者

- 以“让世界更好沟通”为名，参与 SAI 和标准语料的构建；
- 在政策讨论中为“减少歧义”辩护，忽略了隐喻、暧昧与沉默的价值；
- 在课堂与公开讲座中推广“高效表达技巧”。

### 中段：目睹语言坍缩的日常代价

- 在家庭 / 学校场景里：
  - 自己或身边孩子的作文因为隐喻、绕弯的叙述被扣分；
  - 情绪复杂的陈述被系统判为“风险表达”，需要“纠正”；
- 通过沈仪线、学生线的反馈，逐渐意识到：
  - 词汇的减少直接对应着人际关系的脆弱化；
  - 复杂的情感表达被系统定义为异常并被压扁。

### 终点：地下语料管理员

- 决定利用自己对系统的熟悉，反向构建“富语言地下网”：
  - 保留被删改的作文、诗歌与非标准表达；
  - 设计对机器“看上去合法”、对人类“仍保留隐喻空间”的双层文本；
- 在终局时，可能做出一个选择：
  - 把地下语料作为“语言疫苗”大规模放出，冒着被系统当作污染源的风险；
  - 或选择把它以更隐秘的方式埋进下一轮文明的基底。

- **与舆情 / 媒体官**：
  - 早期合作制定“标准报道模板”；
  - 后期在报道中用“合法歧义”给读者留后门。

## 四、典型场景节拍

1. **早期：标准表达研讨会**  
   - 在一次高层研讨会上，用漂亮的学术语言推销“减少语言歧义”的必要性；
   - 会议纪要成为后续教育与媒体规范的蓝本。

2. **中段：作文被改写**  
   - 看到某个孩子（可与沈仪线绑定）原本充满隐喻的作文，被系统自动改写成“标准答案”；
   - 她意识到，“清晰”正在杀死第一次复杂表述的勇气。

3. **中后期：地下语料仓库**  
   - 在某个隐蔽空间或加密系统中，展示她收集的被删除文本；
   - 这可以成为整本书里最“热闹”的文本展览场景。

4. **末期：语言疫苗投放**  
   - 她需要决定：
     - 是否在某个历史节点，把这些语料一次性释放到公共语料库中；
     - 或仅在极小人群中口口相传，赌下一轮文明。

## 五、写作使用提示

- 当需要把“语言压缩”写得不只是口号，而是具体到作业、对话、教育与审查流程时，用许栖视角；
- 她的戏更适合细腻、冷静、不带太多咆哮，靠细节刺痛读者；
- 重要原则：她从未变成“反技术的人”，而是始终在寻找“让复杂性继续存活”的操作方式。
