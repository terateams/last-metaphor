# 重要配角总表｜Supporting Cast

> 用途：集中管理所有 B 层重要配角，确保人数与功能可控；每人只完成 1–2 个关键任务，不给完整弧线。

---

## 一、使用原则与配额

- **总量目标**：反复出现的配角控制在 12–18 人；
- **命名标准**：
  - 有稳定职业 + 在至少两处关键情节出现的角色才给全名；
  - 只承担一次性功能的，尽量使用职位 + 首字母代称即可。
- **功能检查四连问**：
  - 是否推动过一个关键转折？
  - 是否展示过一个系统的真实运作？
  - 是否制造过一次不可逆代价？
  - 是否带来了一个新的视角（阶层 / 职业 / 价值）？

四项皆否：不命名，降级为功能性人物。

---

## 二、按系统分区列席位

下面先列“坑位”，具体姓名可以在动笔时补充。

### 1. 科研 / 技术线

1. **考古队负责人**  
   - 功能：与林峤在“证据链 vs 项目进度”上长期冲突；
   - 典型场景：压报告、删异常、对上级汇报时自动滤干噪声。

2. **材料学专家**  
   - 功能：推动“可复刻”诱惑，主张快速工业化古人技术；
   - 典型场景：展示复刻原型、淡化风险、许诺收益。

3. **AI 布道者 / 产品负责人**  
   - 功能：把“精确”“标准化”包装成个人成功学；
   - 典型场景：发布会、采访、内部动员会，输出金句。

4. **实验室安全官 / 风险评估师**（可选）  
   - 功能：代表“按流程办事”的风险控制视角；
   - 典型场景：在报告上签字，或者拒绝签字。

### 2. 治理 / 舆论线

5. **舆情 / 媒体官**  
   - 功能：维护幸福叙事与官方一致性；
   - 典型场景：删改新闻标题，重写事故报道。

6. **教育系统督导**  
   - 功能：把 SAI 指标嵌入学校考核；
   - 典型场景：巡视课堂、检查教案、下达整改通知。

7. **举报者 / 合规官**  
   - 功能：把恐惧流程化，把人与人之间的不信任变成数据上报；
   - 典型场景：接线举报、审核异常言论、约谈教师或员工。

8. **委员会法学 / 伦理专家**  
   - 功能：在第六部清算时与杜衡、韩策等人进行公开辩驳；
   - 典型场景：听证会、闭门讨论、面向公众的解释发言。

### 3. 日常 / 地下网络线

9. **诗歌黑市中间人**  
   - 功能：运营“语言疫苗”地下渠道，串联许栖与普通人；
   - 典型场景：小书店、非法打印铺、加密聊天室。

10. **旧档案管理员 / 历史修订员**  
    - 功能：负责“优化历史”，亲手删改档案与教材；
    - 典型场景：在档案馆关灯后的独处时刻，决定删哪一页。

11. **医院 / 急救系统管理者**  
    - 功能：将“正确但不可用”的制度具体化；
    - 典型场景：在指标漂亮的报表前，签下一份“事故无责任”的文件。

### 4. 亲缘 / 下一代线

12. **家属 / 伴侣 A（与沈仪绑定）**  
    - 功能：提供“别折腾、先活下去”的现实声音；
    - 典型场景：厨房争吵、孩子教育方向上的冲突。

13. **家属 / 伴侣 B（与罗启或精英线绑定）**  
    - 功能：让高级决策带上非常具体的情感代价；
    - 典型场景：在事故 / 清算前夜的争执或沉默。

14. **年轻学生 / 孩子**（可 1–3 人）  
    - 功能：让语言坍缩与指标化教育的代价可见可痛；
    - 典型场景：作文、口头汇报、被系统“温柔纠正”的瞬间。
