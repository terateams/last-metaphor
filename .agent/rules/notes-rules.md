# `notes/` 目录使用规则

该目录包含小说写作的关键背景和具体指导。**当上下文匹配时，你必须主动查阅这些文件。**

## 1. **场景写作与共情**
- **当写作涉及隐喻、抽象概念或情感落地的场景时：**
  - **必须阅读**：`notes/metaphor-grounding.md`
  - **目标**：应用**“隐喻官僚化”**技法——将抽象隐喻转化为行政现实（表格、罚款、指标）。
- **当写作关于普通人（沈仪、罗启、苏奕）或 SAI 统治下的日常生活时：**
  - **必须阅读**：`notes/ordinary-people-scenes.md`
  - **目标**：聚焦**“日常的疼痛”**以及系统如何具体地干预生活细节。

## 2. **世界观与机制**
- **当讨论 SAI（语义歧义指数）、指标或治理逻辑时：**
  - **参考**：`notes/sai-measurement-paradox.md`（理论支撑与悖论）以及 `notes/corporate-capture-mechanisms.md`（政策演变机制）。
- **当处理“混乱 vs 精确”这一核心母题时：**
  - **参考**：`notes/chaos-vs-precision-meta-theme.md`

## 3. **叙事与结构**
- **当处理角色或代际交接时：**
  - **参考**：`notes/generational-handoff.md`

## 通用写作原则

- **基调**：**“午夜直播间”模式**——批判性、结构性、深挖每个设定的“代价”。
- **微观指标化**：总是寻找方法将人类体验转化为数字/指标（SAI 分数、合规率）。
- **拒绝空洞隐喻**：如果你要使用隐喻，请自问：“在这个世界里，它如何表现为一种物理/行政力量？”（参考 `notes/metaphor-grounding.md`）
